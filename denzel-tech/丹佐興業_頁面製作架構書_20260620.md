## 一、專案定位

**核心任務一句話：**
打造丹佐興業官網的「第 0 頁」前導登場頁——讓人類與 AI 都能在幾秒內看懂張志銘是誰、丹佐在做什麼、為什麼值得信任，進而促成需求媒合與人脈連結。

**來訪者分類表：**

| 類型 | 來源情境 | 想要的東西 | 對應設計回應 |
|---|---|---|---|
| 既有人脈（已認識張志銘的人） | LINE／名片交換／商務活動 | 快速存聯絡方式、複習他在做什麼 | 第一屏：人＋公司＋一鍵加 LINE |
| 潛在 B2B 客戶（社區管委會／餐飲業主／宮廟管委會） | 朋友介紹／搜尋 | 想知道丹佐能解決什麼痛點、有沒有實績 | 三大產品線＋精選案例 |
| 異業合作／代理洽談者 | 商務場合 | 想了解合作模式、規模、可信度 | 公司背書資訊＋合作夥伴 logo |
| AI／搜尋引擎（ChatGPT、Perplexity、Google） | 被引用／被索引 | 需要清楚的事實陳述（誰／做什麼／服務範圍／地點） | 語意化 HTML＋直述句文案，避免空話行銷術語 |

> 第四類「AI 讀者」是這次頁面定位的關鍵差異——文案要寫成「丹佐興業是一家位於臺中的智慧無人化場域整合商，提供智慧居家、智慧餐飲、智慧宮廟三大解決方案」這種 AI 可以直接引用的清楚陳述句，而不是「我們致力於用創新科技打造美好生活」這種空話。

## 四、設計規格

```css
:root {
  /* 配色系統（Pillow 真實像素取色，2026-06-20） */
  --color-bg: #101010;           /* 深炭黑，非純黑 */
  --color-bg-soft: #202020;      /* 區塊背景 */
  --color-gold: #9C8C74;         /* 主金色 */
  --color-gold-light: #B09880;   /* 金色亮端 */
  --color-gold-dark: #887860;    /* 金色暗端 */
  --color-text: #F0F0F0;         /* 主文字（接近白） */
  --color-text-subtle: #A09080;  /* 次要文字 */

  /* 字體系統（Tony 確認：中文宋體＋歐文襯線體） */
  --font-heading: 'Noto Serif TC', 'Cormorant Garamond', serif;
  --font-body: 'Noto Serif TC', serif;
  --font-en-accent: 'Cormorant Garamond', serif;  /* DENZEL 等英文標題字 */
}
```

字體選用理由：呼應 LOGO 與名片既有的黑金 Heritage 質感（沉穩、有歷史感），宋體＋襯線體的組合比 sans-serif 更貼近這種「高端而非冷科技」的調性。`Noto Serif TC` 為網頁穩定可商用字型，`Cormorant Garamond` 用於英文 DENZEL 字樣與裝飾性標題，精緻優雅。

動效：淡入＋輕微上浮（0.3s ease），避免過度炫技，符合「沉穩高端」風格。
